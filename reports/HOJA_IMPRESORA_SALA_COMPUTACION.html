Modelo

Brother DCP-L2540DW series

Nombre en Windows

Brother Sala Computacion

Direccion IP

192.168.0.208

Tipo de conexion

Impresora de red por IP

Como imprimir

  1. Abre tu documento y presiona Ctrl + P.
  2. Selecciona la impresora Brother Sala Computacion.
  3. Revisa que el tamaño de papel sea Carta.
  4. Confirma orientación, cantidad de copias y páginas antes de imprimir.
  5. Si tu archivo viene en oficio, A4 u otro tamaño, ajústalo primero a Carta.

Agregar en Windows

Opcion rapida

  1. Ve a Configuracion > Bluetooth y dispositivos > Impresoras y escaneres.
  2. Haz clic en Agregar dispositivo.
  3. Si no aparece, elige Agregar manualmente.

Opcion manual por IP

  1. Elige Agregar una impresora mediante una direccion TCP/IP.
  2. Direccion o nombre de host: 192.168.0.208.
  3. Nombre sugerido: Brother Sala Computacion.
  4. Si Windows pide controlador, selecciona Brother DCP-L2540DW series.

Antes de imprimir

  • Verifica vista previa.
  • Revisa si es una o doble cara.
  • Imprime solo lo necesario.
  • Si tienes dudas, consulta antes de lanzar muchas páginas.

Atencion

Esta impresora usa SOLO papel tamaño CARTA. Revisa tus documentos antes de imprimir para no desperdiciar hojas.
